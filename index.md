#### 使用本站提供的 Ngrok 服务

* 下载 ngrok 客户端，请根据使用的操作系统，下载对应的版本

* 运行 ngrok.exe

例如

**建立HTTP隧道**

```HTTP
ngrok -subdomain pub -proto=http -config=ngrok.cfg 80
```

**建立数据库隧道**

```
ngrok -proto=tcp -config=ngrok.cfg 3306
```

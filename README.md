# Mgrok

Ngrok 是一个很出名的代理工具，但是从 1.7 以后作者已经不公开源码。但由于 1.7 版本现有的功能，不满足我的需求，因此对它进行修改，并命名为 Mgrok

* [代码的编译](http://www.mgrok.cn/#mgrok_build)

* [使用说明](http://www.mgrok.cn/#usage)
